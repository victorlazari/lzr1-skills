## 1. RevOps Framework

### RevOps Functions

| Function | Responsibility | Impact |
|---|---|---|
| Strategy & Planning | Forecasting, territory, quota | Revenue predictability |
| Process & Methodology | Sales stages, playbooks, rules of engagement | Efficiency |
| Technology | CRM, tools, integrations, data | Productivity |
| Analytics & Insights | Reporting, dashboards, analysis | Decision quality |
| Enablement | Training, content, onboarding | Rep effectiveness |
| Compensation | Plans, SPIFs, accelerators | Motivation and alignment |

### Revenue Model

```
Revenue = Opportunities × Win Rate × Average Deal Size

Pipeline needed = Quota / Win Rate
Activities needed = Pipeline needed / Conversion rates

Example:
  Quota: $1M
  Win rate: 25%
  Pipeline needed: $4M
  Avg deal: $50K
  Opportunities needed: 80
  Meeting-to-opportunity: 30%
  Meetings needed: 267
  Outreach-to-meeting: 3%
  Outreach needed: 8,900
```

## 2. Sales Analytics

### Key Reports

| Report | Audience | Frequency | Content |
|---|---|---|---|
| Pipeline review | Sales managers | Weekly | Stage, velocity, risks |
| Forecast | Leadership | Weekly | Commit, best case, upside |
| Win/Loss analysis | All sales | Monthly | Why we win/lose, patterns |
| Activity metrics | Reps, managers | Daily/Weekly | Calls, emails, meetings |
| Funnel conversion | RevOps, leadership | Monthly | Stage-to-stage rates |
| Rep performance | Managers | Monthly | Quota attainment, trends |
| Cohort analysis | RevOps | Quarterly | Performance by segment, rep tenure |

### Sales Efficiency Metrics

| Metric | Formula | Benchmark |
|---|---|---|
| Magic Number | Net new ARR / Prior quarter S&M spend | >0.75 (efficient) |
| CAC Ratio | S&M cost / New customers | Decreasing over time |
| Quota attainment | Revenue closed / Quota | >70% of reps at 100%+ |
| Ramp time | Time for new rep to hit quota | <6 months |
| Sales productivity | Revenue per rep | Growing |
| Pipeline creation | New pipeline per rep per month | 3-4x monthly quota |

## 3. Compensation Design

### Compensation Structure

| Component | Description | Typical Split |
|---|---|---|
| Base salary | Fixed compensation | 50-60% OTE (enterprise) |
| Variable (commission) | Performance-based pay | 40-50% OTE (enterprise) |
| Accelerators | Above-quota multiplier | 1.5-3x rate above 100% |
| Decelerators | Below-threshold reduction | 0.5x rate below 50% |
| SPIFs | Short-term incentives | Bonus for specific behaviors |
| Bonuses | Discretionary/MBO | Quarterly/annual |

### Commission Structures

| Structure | Description | Best For |
|---|---|---|
| Linear | Fixed % of revenue | Simple, predictable |
| Tiered | Higher % at higher attainment | Motivate overachievement |
| Accelerated | Multiplier above quota | Enterprise, big deals |
| Multi-variable | Weighted across metrics | Complex selling motions |
| Draw | Advance against future commissions | New reps, ramping |

### OTE Benchmarks (US, 2024-2025)

| Role | OTE Range | Base/Variable Split |
|---|---|---|
| SDR/BDR | $70-100K | 70/30 |
| Mid-Market AE | $150-250K | 50/50 |
| Enterprise AE | $250-400K+ | 50/50 |
| Sales Engineer | $150-250K | 70/30 |
| Account Manager | $120-200K | 60/40 |
| VP Sales | $300-500K+ | 60/40 |

## 4. Territory Planning

### Territory Design Principles

| Principle | Description |
|---|---|
| Balanced opportunity | Similar potential across territories |
| Geographic efficiency | Minimize travel, maximize coverage |
| Account fit | Match rep skills to account complexity |
| Growth potential | Consider both current and future revenue |
| Fair distribution | Equitable across the team |
| Clean boundaries | No overlap, no gaps |

### Territory Models

| Model | Description | Best For |
|---|---|---|
| Geographic | By region/city | Field sales, local presence |
| Named accounts | Specific account list | Enterprise, strategic |
| Vertical/Industry | By industry segment | Specialized solutions |
| Company size | By employee count or revenue | Different sales motions |
| Round-robin | Sequential assignment | Inbound, equal distribution |
| Hybrid | Combination of above | Complex organizations |
