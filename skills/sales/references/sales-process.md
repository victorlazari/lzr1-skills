## 1. Sales Methodologies

### Methodology Comparison

| Methodology | Core Concept | Best For |
|---|---|---|
| MEDDIC/MEDDPICC | Metrics, Economic Buyer, Decision Criteria, Decision Process, Identify Pain, Champion, Competition | Enterprise, complex deals |
| SPIN Selling | Situation, Problem, Implication, Need-Payoff questions | Consultative, mid-market |
| Challenger Sale | Teach, Tailor, Take Control | Commoditized markets |
| Sandler | Pain, Budget, Decision (reverse selling) | SMB, transactional |
| Solution Selling | Problem-first, solution-second | Technical products |
| Command of the Message | Value framework, before/after | Platform sales |
| BANT | Budget, Authority, Need, Timeline | Quick qualification |
| Gap Selling | Current state → Future state (gap = value) | Value-based selling |

### MEDDPICC Deep Dive

| Element | Question | Strong Answer |
|---|---|---|
| Metrics | What business outcomes will they measure? | Specific KPIs with targets |
| Economic Buyer | Who signs the check? | Identified, engaged, supportive |
| Decision Criteria | How will they evaluate solutions? | Known, we align well |
| Decision Process | What steps to purchase? | Mapped, timeline confirmed |
| Paper Process | Legal, procurement, security? | Known, no blockers |
| Identify Pain | What's the compelling event? | Urgent, quantified |
| Champion | Who's selling internally for us? | Powerful, motivated, coached |
| Competition | Who else are they evaluating? | Known, differentiated |

## 2. Pipeline Management

### Pipeline Stages

| Stage | Definition | Exit Criteria | Probability |
|---|---|---|---|
| Prospecting | Initial outreach, no response yet | Meeting booked | 5% |
| Discovery | First meeting, understanding needs | Pain confirmed, next steps | 10% |
| Qualification | MEDDIC validated | Champion, budget, timeline | 20% |
| Solution | Demo/POC, solution presented | Technical win | 40% |
| Proposal | Pricing/proposal delivered | Verbal agreement | 60% |
| Negotiation | Terms discussion | Final terms agreed | 80% |
| Closed Won | Contract signed | Revenue recognized | 100% |

### Pipeline Metrics

| Metric | Formula | Healthy Benchmark |
|---|---|---|
| Pipeline coverage | Pipeline value / Quota | 3-4x quota |
| Win rate | Closed-won / Total opportunities | 20-30% (enterprise) |
| Average deal size | Total revenue / Deals closed | Growing over time |
| Sales cycle length | Days from creation to close | Stable or decreasing |
| Stage conversion | Opportunities advancing / Total in stage | >50% per stage |
| Pipeline velocity | (Opportunities × Win rate × ACV) / Cycle days | Growing |

### Forecasting Methods

| Method | Description | Accuracy |
|---|---|---|
| Weighted pipeline | Sum of (deal value × stage probability) | Low-medium |
| Commit/Best case/Upside | Rep categorization | Medium |
| Historical trending | Based on past conversion rates | Medium-high |
| AI/ML forecasting | Pattern-based prediction | High (with data) |
| Bottom-up | Deal-by-deal assessment | High (small pipeline) |

## 3. Discovery and Qualification

### Discovery Call Structure

```
1. Opening (5 min)
   - Agenda, time check, rapport
   - "What would make this a good use of your time?"

2. Situation (10 min)
   - Current state, tools, team, process
   - "Walk me through how you currently handle..."

3. Problem (15 min)
   - Pain points, challenges, impact
   - "What's the biggest challenge with..."
   - "What happens when that goes wrong?"

4. Impact (10 min)
   - Business impact, cost of inaction
   - "What does that cost you in terms of..."
   - "What would it mean if you could solve this?"

5. Next Steps (5 min)
   - Mutual action plan
   - "Based on what you've shared, here's what I'd suggest..."
```

### Qualification Criteria

| Signal | Strong (Pursue) | Weak (Deprioritize) |
|---|---|---|
| Pain | Urgent, quantified, top priority | Nice-to-have, vague |
| Budget | Allocated, approved | No budget, "exploring" |
| Timeline | Defined deadline, compelling event | "Sometime this year" |
| Authority | Economic buyer engaged | No access to decision maker |
| Champion | Internal advocate, power, motivation | Friendly but no influence |
| Fit | Ideal customer profile match | Edge case, poor fit |

## 4. Negotiation and Closing

### Negotiation Principles

| Principle | Description | Tactic |
|---|---|---|
| Never negotiate against yourself | Don't offer discounts unprompted | Wait for them to ask |
| Trade, don't give | Every concession gets something back | "If we do X, would you do Y?" |
| Expand the pie | Find non-price value to trade | Terms, timeline, scope |
| Create urgency | Legitimate reasons to decide now | End-of-quarter, price increase |
| Walk-away power | Be willing to lose the deal | Strong pipeline = confidence |
| Anchor high | First number sets the range | Start with full price |

### Common Objections and Responses

| Objection | Type | Response Framework |
|---|---|---|
| "Too expensive" | Price | Reframe to value/ROI, compare to cost of inaction |
| "We're happy with current solution" | Status quo | Quantify hidden costs, show what they're missing |
| "Need to think about it" | Stall | Identify real concern, offer to address it |
| "Not the right time" | Timing | Find compelling event, quantify delay cost |
| "Need to check with [person]" | Authority | Offer to present together, arm champion |
| "Competitor does X" | Competition | Acknowledge, differentiate on what matters most |

## 5. Outbound Prospecting

### Multi-Channel Sequence

| Day | Channel | Action |
|---|---|---|
| 1 | Email | Personalized value-based email |
| 2 | LinkedIn | Connection request + note |
| 4 | Phone | Call attempt #1 |
| 5 | Email | Follow-up with relevant content |
| 8 | LinkedIn | Engage with their content |
| 10 | Phone | Call attempt #2 |
| 12 | Email | Different angle/trigger |
| 15 | Phone | Call attempt #3 |
| 18 | Email | Breakup email |

### Cold Email Framework

```
Subject: [Specific, relevant, curiosity-driven]

Hi [Name],

[Observation about their company/role — shows research]

[Problem statement — something they likely face]

[Brief credibility — how you've helped similar companies]

[Specific ask — meeting, not a pitch]

[Signature]
```

### Prospecting Metrics

| Metric | Target | Description |
|---|---|---|
| Activities per day | 50-80 | Emails + calls + social touches |
| Response rate | 5-15% | Replies / Outreach sent |
| Meeting rate | 2-5% | Meetings / Outreach sent |
| Show rate | 80%+ | Meetings held / Meetings booked |
| Pipeline generated | 3-4x quota | Value of opportunities created |
