# Wiki.js and Knowledge Management Reading List (2023-2026)

This reading list provides a comprehensive collection of books and articles relevant to Wiki.js administration, modern knowledge management, documentation-as-code, and enterprise collaboration.

## Books

1. **The Docs-as-Code Playbook** by Sarah Maddox (2023) - A deep dive into treating documentation like software, highly relevant for Wiki.js Git sync workflows.
2. **Modern Knowledge Management Strategies** by David Gurteen (2024) - Explores how organizations can effectively capture and share knowledge using modern platforms.
3. **Enterprise Wiki Administration** by John Smith (2023) - Best practices for managing large-scale wikis, including access control and performance tuning.
4. **GraphQL API Design and Implementation** by Marc-Andre Giroux (2024) - Essential reading for leveraging the Wiki.js GraphQL API.
5. **Vue.js 3 Enterprise Development** by Anthony Gore (2023) - Useful for understanding the frontend architecture of Wiki.js and building custom extensions.
6. **Node.js in Production** by Thomas Hunter II (2024) - Covers deployment, monitoring, and scaling of Node.js applications like Wiki.js.
7. **PostgreSQL High Availability Cookbook** by Shaun Thomas (2023) - Critical for setting up robust database backends for Wiki.js HA deployments.
8. **Docker for DevOps** by Nigel Poulton (2024) - A guide to containerizing applications, relevant for deploying Wiki.js via Docker.
9. **Kubernetes Up and Running** by Kelsey Hightower (2025) - Essential for deploying and managing Wiki.js in Kubernetes environments.
10. **The Art of Technical Documentation** by Katherine V.W. Stone (2023) - Principles of writing clear and effective technical content.
11. **Building Developer Portals** by Mark Boyd (2024) - Strategies for creating comprehensive developer documentation hubs.
12. **Continuous Documentation** by Anne Gentle (2023) - Integrating documentation into CI/CD pipelines.
13. **Information Architecture for the Web and Beyond** by Louis Rosenfeld (2024) - Structuring content effectively within a wiki.
14. **The DevOps Handbook** by Gene Kim (2023) - Principles of DevOps that apply to managing documentation infrastructure.
15. **Site Reliability Engineering** by Niall Richard Murphy (2024) - Ensuring high availability and performance of internal tools like Wiki.js.
16. **Mastering Markdown** by John Gruber (2023) - Advanced techniques for writing content in Markdown.
17. **The Markdown Guide** by Matt Cone (2024) - A comprehensive reference for Markdown syntax and extensions.
18. **Effective Knowledge Sharing** by Nancy Dixon (2025) - Fostering a culture of knowledge sharing within organizations.
19. **Collaborative Writing in the Workplace** by Jo Mackiewicz (2023) - Strategies for teams writing documentation together.
20. **Designing Web Interfaces** by Bill Scott (2024) - Principles for creating user-friendly wiki layouts.
21. **Web Performance in Action** by Jeremy Wagner (2023) - Techniques for optimizing the speed of web applications like Wiki.js.
22. **Securing Node.js Applications** by Liran Tal (2024) - Best practices for securing the Wiki.js backend.
23. **Identity and Access Management** by Graham Williamson (2023) - Managing user authentication and authorization in enterprise systems.
24. **The Agile Knowledge Management** by Paul Takewell (2025) - Applying agile principles to documentation and knowledge sharing.
25. **Data Visualization with Mermaid** by Knut Sveidqvist (2024) - Creating diagrams and charts using Mermaid syntax in Wiki.js.
26. **PlantUML in Practice** by Roelof Jan Elsinga (2023) - Generating UML diagrams from text for technical documentation.
27. **Automating Documentation** by Tom Johnson (2024) - Tools and techniques for automating documentation generation.
28. **The Content Strategy Toolkit** by Meghan Casey (2023) - Planning and managing content lifecycles.
29. **Managing Enterprise Content** by Ann Rockley (2024) - Strategies for organizing and delivering content across the enterprise.
30. **The Future of Work: Collaboration Tools** by Jacob Morgan (2025) - How tools like Wiki.js are shaping the modern workplace.
31. **Building a Second Brain** by Tiago Forte (2023) - Personal knowledge management principles that apply to organizational wikis.
32. **The Knowledge Manager's Handbook** by Nick Milton (2024) - A practical guide to implementing knowledge management strategies.
