## 1. Corporate Governance

### Board of Directors

| Aspect | Description |
|---|---|
| Composition | Founders, investors, independents |
| Fiduciary duties | Duty of care, duty of loyalty |
| Meetings | Quarterly (minimum), special as needed |
| Committees | Audit, compensation, nominating/governance |
| Minutes | Formal record of decisions and discussions |
| Consent actions | Written resolutions without meeting |

### Corporate Actions Requiring Board Approval

| Action | Board | Stockholder |
|---|---|---|
| Equity issuance | Yes | Sometimes (depending on charter) |
| Fundraising | Yes | Sometimes |
| M&A / Sale of company | Yes | Yes |
| Annual budget | Yes | No |
| Executive compensation | Yes | No (advisory vote if public) |
| Dividends | Yes | No |
| Bylaws amendment | Yes | Sometimes |
| Charter amendment | Yes | Yes |

## 2. Entity Management

### Entity Types for Startups

| Entity | Tax Treatment | Liability | Best For |
|---|---|---|---|
| C-Corp (Delaware) | Double taxation, but preferred by VCs | Limited | VC-backed startups |
| S-Corp | Pass-through | Limited | Small businesses, no VC |
| LLC | Flexible (pass-through default) | Limited | Consulting, small projects |
| B-Corp | C-Corp + social mission | Limited | Mission-driven companies |

### Delaware C-Corp Setup

```
Formation documents:
1. Certificate of Incorporation (filed with DE Secretary of State)
2. Bylaws (internal governance rules)
3. Initial board resolutions (officers, bank, fiscal year)
4. Stock purchase agreements (founder shares)
5. IP assignment agreements (founders assign IP to company)
6. 83(b) elections (if restricted stock)
7. Qualify to do business in operating state(s)
```

## 3. Intellectual Property

### IP Types

| Type | Protects | Duration | Registration |
|---|---|---|---|
| Patent | Inventions, processes | 20 years from filing | Required (USPTO) |
| Copyright | Original works of authorship | Life + 70 years (or 95 years for work-for-hire) | Automatic (registration strengthens) |
| Trademark | Brand names, logos, slogans | Indefinite (with use) | Recommended (USPTO) |
| Trade secret | Confidential business information | Indefinite (while secret) | No registration (protection measures) |

### IP Protection Strategy

| Strategy | Implementation |
|---|---|
| Employee IP assignment | All employees sign IP assignment agreements |
| Contractor IP assignment | All contractors assign work product |
| Invention disclosure | Process for employees to report inventions |
| Patent filing | File provisional/utility patents for key innovations |
| Trademark registration | Register brand names and logos |
| Trade secret program | Identify, mark, and protect confidential information |
| Open source policy | Clear rules on using and contributing to OSS |
| Non-compete/Non-solicit | Where enforceable, protect key relationships |

## 4. Open Source Compliance

### License Categories

| Category | Licenses | Obligation | Risk Level |
|---|---|---|---|
| Permissive | MIT, BSD, Apache 2.0 | Attribution only | Low |
| Weak copyleft | LGPL, MPL | Share modifications to the library | Medium |
| Strong copyleft | GPL, AGPL | Share entire derivative work | High |
| Network copyleft | AGPL | Share if accessed over network | Very High |
| Creative Commons | CC-BY, CC-BY-SA | Varies by variant | Low-Medium |

### Open Source Policy Framework

| Area | Policy |
|---|---|
| Using OSS | Approved license list, scanning in CI/CD |
| Contributing to OSS | CLA/DCO, approval process, IP review |
| Releasing OSS | Business case, license selection, IP clearance |
| Copyleft compliance | Isolation strategies, legal review triggers |
| SBOM | Software Bill of Materials maintained |
| Vulnerability management | Monitor for CVEs in dependencies |

## 5. Employment Law Essentials

### Key Employment Considerations

| Area | US Consideration | Risk |
|---|---|---|
| At-will employment | Default in most states (except Montana) | Wrongful termination claims |
| Classification | Employee vs independent contractor | Misclassification penalties |
| Non-competes | Increasingly restricted (FTC ban pending) | Enforceability varies by state |
| Wage and hour | FLSA exempt vs non-exempt | Overtime liability |
| Discrimination | Title VII, ADA, ADEA | Litigation, damages |
| Harassment | Hostile work environment, quid pro quo | Liability, reputation |
| Leave | FMLA, state laws, ADA accommodation | Compliance complexity |
| Termination | Documentation, final pay, WARN Act | Legal exposure |

### Employee vs Contractor

| Factor | Employee | Contractor |
|---|---|---|
| Control | Company controls how work is done | Controls own methods |
| Schedule | Set by company | Self-determined |
| Tools | Company-provided | Own tools |
| Exclusivity | Works primarily for one company | Multiple clients |
| Integration | Integral to business | Supplemental |
| Duration | Ongoing relationship | Project-based |
| Benefits | Entitled to benefits | No benefits |
| Taxes | Company withholds | Self-employment tax |
