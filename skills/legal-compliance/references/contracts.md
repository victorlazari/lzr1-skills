## 1. Contract Types

### Common Technology Contracts

| Contract | Purpose | Key Terms |
|---|---|---|
| SaaS/Subscription Agreement | Software access license | SLA, data handling, termination |
| Master Service Agreement (MSA) | Framework for ongoing services | Liability, IP, payment terms |
| Statement of Work (SOW) | Specific project scope under MSA | Deliverables, timeline, fees |
| NDA (Mutual/One-way) | Protect confidential information | Definition, term, exclusions |
| Data Processing Agreement (DPA) | GDPR data processor obligations | Sub-processors, transfers, deletion |
| Reseller/Partner Agreement | Channel distribution | Margins, territories, obligations |
| Employment Agreement | Employee terms | Compensation, IP assignment, non-compete |
| Independent Contractor Agreement | Contractor engagement | Scope, IP ownership, classification |
| Terms of Service (ToS) | End-user terms | Acceptable use, liability, disputes |
| Privacy Policy | Data collection disclosure | Data types, purposes, rights |

## 2. Key Contract Clauses

### Critical Clauses

| Clause | Purpose | Negotiation Points |
|---|---|---|
| Limitation of liability | Cap on damages | Cap amount (typically 12 months fees) |
| Indemnification | Protection from third-party claims | Scope, IP indemnity, data breach |
| Termination | How to end the agreement | For cause, for convenience, cure period |
| IP ownership | Who owns what | Background IP, developed IP, licenses |
| Confidentiality | Protect sensitive information | Definition, term, exceptions |
| Data protection | Data handling obligations | Processing, security, breach notification |
| SLA | Performance guarantees | Uptime %, credits, measurement |
| Payment terms | When and how to pay | Net 30/60, late fees, disputes |
| Governing law | Which jurisdiction applies | State/country, venue |
| Force majeure | Excuses for non-performance | Definition, notification, duration |
| Assignment | Transfer of contract | Consent required, change of control |
| Warranty | Quality guarantees | Scope, disclaimers, remedies |

### SLA Structure

| Component | Description | Example |
|---|---|---|
| Uptime commitment | Guaranteed availability | 99.9% monthly uptime |
| Measurement | How uptime is calculated | (Total minutes - Downtime) / Total minutes |
| Exclusions | What doesn't count | Scheduled maintenance, force majeure |
| Credits | Remedy for breach | 10% credit per 0.1% below SLA |
| Credit cap | Maximum credits | 30% of monthly fees |
| Claim process | How to request credits | Written notice within 30 days |

## 4. Negotiation Strategy

### Negotiation Priorities by Role

| Your Role | Priority | Approach |
|---|---|---|
| Vendor (selling) | Limit liability, protect IP, ensure payment | Start with your paper, concede strategically |
| Customer (buying) | Data protection, SLA, termination flexibility | Push for customer-friendly terms |
| Partner | Fair economics, clear responsibilities | Balanced, long-term relationship focus |

### Common Negotiation Tradeoffs

| If They Want... | You Can Offer... | In Exchange For... |
|---|---|---|
| Lower price | Volume discount | Longer term, upfront payment |
| Higher liability cap | Uncapped for IP/confidentiality | Mutual cap, insurance requirement |
| Termination for convenience | 30-day notice | Minimum commitment period |
| Custom SLA | Higher uptime guarantee | Premium pricing tier |
| Unlimited indemnity | Broad indemnity scope | Carve-outs for customer actions |

## 5. Contract Lifecycle Management

### CLM Process

| Phase | Activities | Tools |
|---|---|---|
| Request | Business need identified, intake form | Jira, intake portal |
| Draft | Template selection, first draft | CLM platform, templates |
| Review | Legal review, redline | Word, CLM platform |
| Negotiate | Back-and-forth with counterparty | Email, redline tracking |
| Approve | Internal approval workflow | CLM platform, e-signature |
| Execute | Signature, countersignature | DocuSign, PandaDoc |
| Store | Central repository, metadata | CLM platform, SharePoint |
| Monitor | Renewals, obligations, expirations | CLM platform, calendar |

### Contract Metrics

| Metric | Target | Description |
|---|---|---|
| Cycle time | <14 days (standard) | Request to execution |
| Negotiation rounds | <3 rounds | Redline exchanges |
| Template usage | >80% | % using approved templates |
| Renewal rate | >90% | Contracts renewed vs expired |
| Obligation compliance | 100% | Meeting contractual obligations |
