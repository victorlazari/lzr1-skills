## 1. Messaging Framework

### Positioning Statement

```
For [target audience]
Who [need/problem]
[Product name] is a [category]
That [key benefit]
Unlike [alternatives]
We [key differentiator]
```

### Value Proposition Canvas

| Component | Description | Example |
|---|---|---|
| Customer jobs | What they're trying to do | "Deploy code faster" |
| Pains | Frustrations and obstacles | "Manual deployments take hours" |
| Gains | Desired outcomes | "Ship features daily" |
| Pain relievers | How you solve pains | "One-click deployments" |
| Gain creators | How you create gains | "Automated CI/CD pipeline" |
| Products/Services | What you offer | "Deployment platform" |

### Messaging Hierarchy

| Level | Content | Audience |
|---|---|---|
| Tagline | 5-8 words, memorable | Everyone |
| Value proposition | 1-2 sentences, key benefit | Prospects |
| Elevator pitch | 30 seconds, problem + solution | Networking |
| Key messages | 3-5 supporting points | Sales, marketing |
| Proof points | Data, testimonials, case studies | Decision-makers |

## 2. Copywriting Formulas

### Classic Formulas

| Formula | Structure | Best For |
|---|---|---|
| AIDA | Attention → Interest → Desire → Action | Landing pages, ads |
| PAS | Problem → Agitate → Solution | Email, blog intros |
| BAB | Before → After → Bridge | Case studies, sales |
| 4Ps | Promise → Picture → Proof → Push | Sales pages |
| FAB | Feature → Advantage → Benefit | Product descriptions |
| STAR | Situation → Task → Action → Result | Case studies |

### Headline Formulas

```
How to [achieve desired outcome] without [common pain]
[Number] ways to [achieve goal] in [timeframe]
Why [common approach] is [wrong/outdated] (and what to do instead)
The [adjective] guide to [topic] for [audience]
[Do this] to [get this result]
Stop [bad thing]. Start [good thing].
What [successful people] know about [topic] that you don't
```

## 3. Landing Page Copy

### Landing Page Structure

| Section | Purpose | Copy Elements |
|---|---|---|
| Hero | Capture attention, state value | Headline, subhead, CTA |
| Problem | Empathize with pain | Pain points, relatable scenarios |
| Solution | Introduce your product | How it works, key features |
| Benefits | Why it matters | Outcomes, not features |
| Social proof | Build trust | Testimonials, logos, stats |
| Features | Detail capabilities | Feature list with benefits |
| Pricing | Enable decision | Plans, comparison |
| FAQ | Remove objections | Common concerns addressed |
| CTA | Drive action | Final push, urgency |

### CTA Best Practices

| Practice | Example |
|---|---|
| Action-oriented verb | "Start building" not "Submit" |
| Benefit-focused | "Get faster deploys" not "Sign up" |
| Specific | "Start 14-day free trial" not "Try it" |
| Urgency (when genuine) | "Limited spots available" |
| Low friction | "No credit card required" |
| Visible | Contrasting color, above fold |

## 4. Email Copy

### Email Types

| Type | Purpose | Key Elements |
|---|---|---|
| Welcome | Onboard new users | Value reminder, first step |
| Nurture | Build relationship | Educational content, stories |
| Product update | Announce features | What's new, how to use |
| Re-engagement | Win back inactive | Reminder of value, incentive |
| Transactional | Confirm actions | Clear info, next steps |
| Sales | Drive conversion | Problem, solution, CTA |

### Email Structure

```
Subject line: [Curiosity/benefit hook, <50 chars]
Preview text: [Extends subject, <90 chars]

Opening: [Hook — question, stat, or story]
Body: [Value — education, insight, or offer]
CTA: [Single, clear action]
PS: [Optional reinforcement]
```

### Subject Line Formulas

| Formula | Example |
|---|---|
| Question | "Are you still deploying manually?" |
| How-to | "How to cut deploy time by 80%" |
| Number | "3 mistakes killing your CI/CD" |
| Curiosity gap | "We almost deleted production..." |
| Personal | "[Name], your pipeline is ready" |
| Urgency | "Last day: 50% off annual plans" |

## 5. Brand Voice

### Voice Attributes

| Attribute | Description | Do | Don't |
|---|---|---|---|
| Confident | Know your stuff | "Here's how" | "Maybe try..." |
| Helpful | Genuinely useful | "This will save you time" | "Buy now!" |
| Clear | Easy to understand | "Deploy in one click" | "Leverage synergies" |
| Human | Conversational | "We get it" | "The user shall..." |

### Tone Spectrum

| Context | Tone | Example |
|---|---|---|
| Error message | Empathetic, helpful | "Something went wrong. Here's how to fix it." |
| Success | Celebratory, brief | "You're all set! Your deploy is live." |
| Onboarding | Encouraging, clear | "Great choice! Let's get you started." |
| Pricing | Transparent, confident | "Simple pricing. No surprises." |
| Security incident | Serious, transparent | "What happened, what we did, what's next." |

### Style Guide Essentials

| Element | Standard |
|---|---|
| Product name | Always capitalized, never abbreviated |
| Technical terms | Define on first use, then use freely |
| Contractions | Use them (we're, you'll, it's) |
| Oxford comma | Always use |
| Numbers | Spell out 1-9, numerals for 10+ |
| Dates | Month Day, Year (January 15, 2024) |
| Acronyms | Spell out first use, then abbreviate |
