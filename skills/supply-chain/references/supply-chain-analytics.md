## 1. Spend Analysis

### Spend Analysis Process

| Step | Activity | Output |
|---|---|---|
| Collect | Gather all spend data | Raw spend data |
| Cleanse | Normalize, deduplicate | Clean dataset |
| Classify | Categorize by taxonomy | Classified spend |
| Analyze | Identify patterns, opportunities | Insights report |
| Act | Implement savings initiatives | Cost reduction |

### Spend Categories (Kraljic Matrix)

| Quadrant | Characteristics | Strategy |
|---|---|---|
| Strategic | High value, high risk | Partnership, collaboration |
| Leverage | High value, low risk | Competitive bidding, consolidation |
| Bottleneck | Low value, high risk | Secure supply, find alternatives |
| Routine | Low value, low risk | Automate, simplify |

### Cost Reduction Strategies

| Strategy | Typical Savings | Effort |
|---|---|---|
| Volume consolidation | 5-15% | Medium |
| Competitive bidding | 10-20% | Medium |
| Specification optimization | 5-10% | High |
| Process automation | 15-30% (process cost) | High |
| Payment term optimization | 2-5% | Low |
| License right-sizing | 10-30% | Low |
| Contract renegotiation | 5-15% | Medium |

## 2. Demand Planning

### Demand Forecasting Methods

| Method | Description | Best For |
|---|---|---|
| Historical trend | Extrapolate past patterns | Stable demand |
| Moving average | Average of recent periods | Smoothing fluctuations |
| Exponential smoothing | Weighted recent data | Trending demand |
| Regression | Correlate with drivers | Causal relationships |
| Machine learning | Pattern recognition | Complex, multi-factor |
| Collaborative | Input from stakeholders | New products, events |

### Demand Planning Process

| Phase | Activities | Frequency |
|---|---|---|
| Data collection | Gather historical data, market signals | Ongoing |
| Statistical forecast | Generate baseline forecast | Monthly |
| Demand sensing | Incorporate real-time signals | Weekly |
| Consensus planning | Cross-functional alignment | Monthly |
| Adjustment | Apply business intelligence | As needed |
| Accuracy tracking | Measure forecast vs actual | Monthly |

## 3. Inventory Management

### Inventory Models

| Model | Description | Best For |
|---|---|---|
| Just-in-time (JIT) | Minimize inventory, order as needed | Stable supply |
| Safety stock | Buffer for uncertainty | Variable demand/supply |
| Economic order quantity | Optimize order size | Regular purchases |
| Vendor-managed | Supplier manages inventory | Trusted partnerships |
| Consignment | Pay only when used | High-value items |

### Inventory Metrics

| Metric | Formula | Target |
|---|---|---|
| Inventory turnover | COGS / Average inventory | Higher is better |
| Days of supply | Inventory / Daily demand | Depends on category |
| Fill rate | Orders filled / Total orders | >95% |
| Stockout rate | Stockouts / Total SKUs | <5% |
| Carrying cost | Storage + Insurance + Depreciation | Minimize |
| Dead stock | Unused items / Total inventory | <5% |

## 4. Supply Chain Optimization

### Optimization Areas

| Area | Techniques | Impact |
|---|---|---|
| Network design | Facility location, routing | 10-20% cost reduction |
| Transportation | Route optimization, consolidation | 5-15% cost reduction |
| Inventory | Safety stock optimization | 10-30% inventory reduction |
| Procurement | Strategic sourcing, auctions | 5-20% cost reduction |
| Production | Scheduling, capacity planning | 10-25% efficiency gain |
| Demand | Forecast accuracy improvement | 10-20% inventory reduction |

### Supply Chain Resilience

| Strategy | Description |
|---|---|
| Diversification | Multiple suppliers per category |
| Nearshoring | Closer suppliers for shorter lead times |
| Buffer stock | Strategic inventory for critical items |
| Visibility | Real-time tracking and monitoring |
| Flexibility | Agile contracts, quick switching |
| Scenario planning | Prepare for disruption scenarios |

## 5. Performance Metrics

### Supply Chain KPIs

| Category | Metric | Target |
|---|---|---|
| Cost | Total supply chain cost as % of revenue | <10% |
| Cost | Procurement savings vs baseline | >5% annually |
| Speed | Order-to-delivery cycle time | Decreasing |
| Speed | Supplier lead time | Decreasing |
| Quality | Supplier defect rate | <1% |
| Quality | On-time delivery rate | >95% |
| Flexibility | Time to onboard new supplier | <30 days |
| Sustainability | Supplier sustainability score | Increasing |

### Supplier Scorecard

| Dimension | Metrics | Weight |
|---|---|---|
| Quality | Defect rate, compliance | 25% |
| Delivery | On-time, lead time | 25% |
| Cost | Price competitiveness, TCO | 25% |
| Responsiveness | Communication, issue resolution | 15% |
| Innovation | New ideas, continuous improvement | 10% |
