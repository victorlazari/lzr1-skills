## 1. Org Design

### Org Structure Models

| Model | Description | Best For |
|---|---|---|
| Functional | Grouped by function (eng, sales, etc.) | Early stage, specialization |
| Divisional | Grouped by product/market/geography | Multi-product, scale |
| Matrix | Dual reporting (function + product) | Complex organizations |
| Flat | Minimal hierarchy | Small teams, autonomy |
| Team-based | Cross-functional squads | Product development |
| Network | Fluid, project-based | Innovation, consulting |

### Span of Control

| Level | Typical Span | Rationale |
|---|---|---|
| IC (Individual Contributor) | N/A | No direct reports |
| Team Lead | 3-5 | Player-coach, hands-on |
| Manager | 5-8 | Full-time management |
| Senior Manager | 6-10 | Experienced, mature teams |
| Director | 5-8 (managers) | Managing managers |
| VP | 4-7 (directors) | Strategic leadership |

### Team Topologies (for Engineering)

| Topology | Description | Purpose |
|---|---|---|
| Stream-aligned | End-to-end product ownership | Value delivery |
| Platform | Internal platform services | Reduce cognitive load |
| Enabling | Coaching and capability building | Skill development |
| Complicated subsystem | Deep specialist expertise | Complex domains |

## 2. Career Frameworks

### Engineering Ladder Example

| Level | Title | Scope | Years (typical) |
|---|---|---|---|
| L1 | Junior Engineer | Task-level, guided | 0-2 |
| L2 | Engineer | Feature-level, independent | 2-4 |
| L3 | Senior Engineer | Project-level, leads others | 4-7 |
| L4 | Staff Engineer | Multi-project, org influence | 7-12 |
| L5 | Principal Engineer | Company-wide impact | 12+ |
| L6 | Distinguished Engineer | Industry impact | 15+ |

### Career Framework Dimensions

| Dimension | Description | Progression |
|---|---|---|
| Technical skill | Depth and breadth of expertise | Increasing complexity |
| Scope | Size of problems tackled | Task → Team → Org → Company |
| Impact | Business outcomes delivered | Individual → Team → Org |
| Leadership | Influence and mentorship | Self → Others → Organization |
| Communication | Clarity and audience | Team → Cross-team → External |
| Autonomy | Independence of work | Guided → Independent → Directing |

### IC vs Management Track

| Aspect | IC Track | Management Track |
|---|---|---|
| Focus | Technical depth, architecture | People, process, strategy |
| Output | Code, designs, technical decisions | Team output, hiring, culture |
| Growth | Deeper expertise, broader influence | Larger teams, bigger scope |
| Meetings | 20-40% of time | 50-70% of time |
| Evaluation | Technical impact, mentorship | Team performance, retention |

## 3. Performance Management

### Performance Review Cycle

| Timing | Activity | Duration |
|---|---|---|
| Ongoing | 1:1s, feedback, goal tracking | Weekly |
| Quarterly | Goal check-in, calibration | 1-2 weeks |
| Semi-annual | Formal review, rating, feedback | 3-4 weeks |
| Annual | Compensation review, promotion | 4-6 weeks |

### Performance Rating Scale

| Rating | Description | Distribution (typical) |
|---|---|---|
| Exceptional | Consistently exceeds, outsized impact | 5-10% |
| Exceeds | Frequently exceeds expectations | 20-25% |
| Meets | Consistently meets expectations | 50-60% |
| Developing | Partially meets, needs improvement | 10-15% |
| Below | Does not meet expectations | 0-5% |

### Goal Setting (OKR Framework)

```
Objective: Qualitative, inspiring goal
  Key Result 1: Quantitative measure (metric + target)
  Key Result 2: Quantitative measure
  Key Result 3: Quantitative measure

Example:
Objective: Become the go-to platform for mid-market companies
  KR1: Increase mid-market ARR from $2M to $5M
  KR2: Achieve NPS of 50+ in mid-market segment
  KR3: Launch 3 features from mid-market customer feedback
```

## 4. Learning and Development

### Development Methods (70-20-10)

| Method | % of Learning | Examples |
|---|---|---|
| On-the-job (70%) | Experience | Stretch assignments, projects, rotations |
| Social (20%) | Others | Mentoring, coaching, peer learning |
| Formal (10%) | Training | Courses, conferences, certifications |

### Manager Development Program

| Module | Content | Duration |
|---|---|---|
| New manager foundations | Role transition, expectations | 2 days |
| Feedback and coaching | Giving feedback, coaching conversations | 1 day |
| Performance management | Reviews, PIPs, difficult conversations | 1 day |
| Hiring and team building | Interviewing, onboarding, team dynamics | 1 day |
| Strategic thinking | Planning, prioritization, communication | 1 day |
| Advanced leadership | Influence, change management, culture | Ongoing |

## 5. Culture and Engagement

### Engagement Drivers

| Driver | Description | Measurement |
|---|---|---|
| Purpose | Connection to company mission | Survey: "My work matters" |
| Growth | Learning and career development | Survey: "I'm growing here" |
| Autonomy | Control over how work is done | Survey: "I have autonomy" |
| Belonging | Inclusion and psychological safety | Survey: "I belong here" |
| Recognition | Appreciation for contributions | Survey: "I feel recognized" |
| Manager | Quality of direct manager | Survey: "My manager supports me" |
| Compensation | Fair pay and benefits | Survey: "I'm fairly compensated" |

### Employee Survey Cadence

| Type | Frequency | Length | Purpose |
|---|---|---|---|
| Annual engagement | Yearly | 40-60 questions | Comprehensive baseline |
| Pulse survey | Monthly/Quarterly | 5-10 questions | Track trends, spot issues |
| Onboarding survey | 30/60/90 days | 10-15 questions | New hire experience |
| Exit survey | At departure | 15-20 questions | Understand attrition |
| Event-based | After changes | 5-10 questions | Change impact |

### Retention Strategies

| Strategy | Description | Impact |
|---|---|---|
| Competitive compensation | Market-rate pay, equity refresh | High |
| Career growth | Clear paths, promotions, development | High |
| Manager quality | Train managers, address poor managers | High |
| Flexibility | Remote/hybrid, flexible hours | High |
| Recognition | Regular, specific, public and private | Medium |
| Culture | Values-aligned, inclusive, fun | Medium |
| Mission | Meaningful work, impact | Medium |
