## 1. Brand Foundations

### Brand Strategy Components

| Component | Description | Output |
|---|---|---|
| Purpose | Why the brand exists beyond profit | Purpose statement |
| Vision | Where the brand is going | Vision statement |
| Mission | How the brand achieves its purpose | Mission statement |
| Values | What the brand stands for | 3-5 core values |
| Personality | How the brand behaves and communicates | Brand archetypes, traits |
| Promise | What customers can always expect | Brand promise statement |

### Brand Architecture

| Model | Description | Example |
|---|---|---|
| Branded house | One master brand for everything | Google, Virgin, Apple |
| House of brands | Independent brands, parent hidden | P&G (Tide, Gillette, Pampers) |
| Endorsed brands | Sub-brands endorsed by parent | Marriott (Courtyard by Marriott) |
| Hybrid | Mix of approaches | Microsoft (Office, Xbox, LinkedIn) |

## 2. Brand Positioning

### Positioning Framework

```
For [target audience]
Who [need/desire]
[Brand] is the [category]
That [key benefit/differentiator]
Because [reason to believe]
Unlike [competitive alternative]
We [unique value]
```

### Positioning Strategies

| Strategy | Description | When to Use |
|---|---|---|
| Category leader | Best in existing category | Market leader position |
| Challenger | Reframe the category | Differentiated approach |
| Category creator | Define new category | Novel solution |
| Niche specialist | Own a specific segment | Focused, underserved market |
| Value | Best price-to-value ratio | Price-sensitive market |
| Premium | Superior quality/experience | Affluent/quality-focused market |

### Competitive Positioning Map

```
High Price
    │
    │   Premium        Luxury
    │   (Quality)      (Status)
    │
────┼──────────────────────── High Differentiation
    │
    │   Economy        Value
    │   (Basic)        (Best deal)
    │
Low Price
```

## 3. Brand Identity

### Visual Identity System

| Element | Description | Specifications |
|---|---|---|
| Logo | Primary brand mark | Full color, mono, reversed, minimum size |
| Color palette | Brand colors | Primary, secondary, accent, neutral |
| Typography | Brand typefaces | Headlines, body, UI, hierarchy |
| Imagery | Photo/illustration style | Style, subjects, treatment |
| Iconography | Icon style and system | Style, grid, usage |
| Layout | Composition principles | Grid, spacing, hierarchy |
| Motion | Animation style | Timing, easing, personality |

### Brand Voice

| Dimension | Spectrum | Example |
|---|---|---|
| Formal ↔ Casual | Where on the spectrum | "We're pleased to announce" vs "Big news!" |
| Serious ↔ Playful | Tone of communication | Technical vs conversational |
| Respectful ↔ Irreverent | Attitude toward conventions | Traditional vs disruptive |
| Enthusiastic ↔ Matter-of-fact | Energy level | Exclamatory vs informational |

## 4. Brand Messaging

### Messaging Hierarchy

| Level | Audience | Content |
|---|---|---|
| Brand narrative | Everyone | Overarching story and purpose |
| Value propositions | Market segments | Segment-specific benefits |
| Product messaging | Buyers/users | Feature-benefit statements |
| Proof points | Skeptics | Evidence, data, testimonials |
| Tagline/slogan | Mass audience | Memorable encapsulation |

### Messaging Framework Template

```
Audience: [Who]
Pain point: [What problem they face]
Desired outcome: [What they want to achieve]
Our solution: [How we help]
Key benefit: [Primary value delivered]
Proof point: [Evidence it works]
CTA: [What we want them to do next]
```

## 5. Brand Management

### Brand Guidelines Contents

| Section | Includes |
|---|---|
| Brand story | Purpose, mission, values, personality |
| Visual identity | Logo, colors, typography, imagery |
| Voice and tone | Writing style, do/don't examples |
| Applications | Business cards, presentations, digital, print |
| Co-branding | Partnership guidelines, logo placement |
| Social media | Platform-specific guidelines |

### Brand Health Metrics

| Metric | Description | Measurement |
|---|---|---|
| Brand awareness | % who know the brand | Surveys, search volume |
| Brand consideration | % who would consider buying | Surveys |
| Brand preference | % who prefer over alternatives | Surveys, market share |
| Brand loyalty | Repeat purchase, advocacy | NPS, retention, referral |
| Brand equity | Financial value of brand | Revenue premium, valuation |
| Share of voice | Brand mentions vs competitors | Social listening, media monitoring |
