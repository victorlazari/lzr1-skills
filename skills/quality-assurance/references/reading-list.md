# Quality Assurance Recommended Reading (2024-2026)

## Essential Books (32)

### Testing Fundamentals
1. **Lessons Learned in Software Testing** — Kaner, Bach & Pettichord (2001), Wiley.
2. **Agile Testing** — Lisa Crispin & Janet Gregory (2009), Addison-Wesley.
3. **More Agile Testing** — Lisa Crispin & Janet Gregory (2014), Addison-Wesley.
4. **Explore It!** — Elisabeth Hendrickson (2013), Pragmatic Bookshelf. Exploratory testing.
5. **The Art of Software Testing** — Glenford Myers (3rd ed, 2011), Wiley.
6. **A Practitioner's Guide to Software Test Design** — Lee Copeland (2004), Artech House.
7. **Perfect Software and Other Illusions About Testing** — Gerald Weinberg (2008).

### Test Automation
8. **Test Automation University** — Angie Jones (online, 2024). Free courses.
9. **Playwright End-to-End Testing** — various (2024). Modern E2E testing.
10. **xUnit Test Patterns** — Gerard Meszaros (2007), Addison-Wesley. Test design patterns.
11. **Growing Object-Oriented Software, Guided by Tests** — Freeman & Pryce (2009), Addison-Wesley.
12. **The Art of Unit Testing** — Roy Osherove (3rd ed, 2024), Manning.
13. **Working Effectively with Legacy Code** — Michael Feathers (2004), Prentice Hall.
14. **Test-Driven Development** — Kent Beck (2002), Addison-Wesley.

### Performance Testing
15. **Performance Testing Guidance for Web Applications** — Microsoft patterns (2007).
16. **The Art of Capacity Planning** — John Allspaw (2nd ed, 2017), O'Reilly.
17. **Systems Performance** — Brendan Gregg (2nd ed, 2020), Addison-Wesley.
18. **Web Performance in Action** — Jeremy Wagner (2016), Manning.
19. **High Performance Browser Networking** — Ilya Grigorik (2013), O'Reilly.

### API Testing
20. **Testing Web APIs** — Mark Winteringham (2022), Manning.
21. **API Testing and Development with Postman** — Dave Westerveld (2021), Packt.

### Security Testing
22. **The Web Application Hacker's Handbook** — Stuttard & Pinto (2nd ed, 2011), Wiley.
23. **OWASP Testing Guide** — OWASP (v4.2, 2023). Web security testing.
24. **Bug Bounty Bootcamp** — Vickie Li (2021), No Starch Press.

### Quality Engineering
25. **Continuous Delivery** — Jez Humble & David Farley (2010), Addison-Wesley.
26. **Accelerate** — Forsgren, Humble & Kim (2018), IT Revolution. DevOps metrics.
27. **Release It!** — Michael Nygard (2nd ed, 2018), Pragmatic Bookshelf.
28. **Site Reliability Engineering** — Google (2016), O'Reilly. SRE practices.
29. **Observability Engineering** — Majors, Fong-Jones & Miranda (2022), O'Reilly.

### Mobile Testing
30. **Hands-On Mobile App Testing** — Daniel Knott (2015), Addison-Wesley.
31. **Mobile Testing** — various (2024). Modern mobile QA approaches.

### AI in Testing
32. **AI-Powered Testing** — various (2024). AI/ML applications in QA.
