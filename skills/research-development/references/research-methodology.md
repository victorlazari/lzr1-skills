## 1. Research Process

### Research Lifecycle

| Phase | Activities | Output |
|---|---|---|
| Question | Define research question, hypothesis | Research proposal |
| Review | Literature review, state of the art | Literature synthesis |
| Design | Methodology, experiment design | Research plan |
| Collect | Data gathering, experiments | Raw data |
| Analyze | Statistical analysis, interpretation | Findings |
| Communicate | Papers, reports, presentations | Publications |
| Apply | Implementation, technology transfer | Products/features |

### Research Question Formulation

| Component | Description | Example |
|---|---|---|
| Population | Who/what is being studied | "Large language models" |
| Intervention | What's being tested | "Chain-of-thought prompting" |
| Comparison | Against what baseline | "vs. zero-shot prompting" |
| Outcome | What's measured | "Accuracy on reasoning tasks" |
| Context | Under what conditions | "On mathematical problems" |

## 2. Literature Review

### Systematic Review Process

| Step | Activity | Tools |
|---|---|---|
| 1. Scope | Define search terms, databases | Research question |
| 2. Search | Query academic databases | Google Scholar, Semantic Scholar, arXiv |
| 3. Screen | Filter by title/abstract | Inclusion/exclusion criteria |
| 4. Extract | Pull key data from papers | Data extraction form |
| 5. Synthesize | Identify themes, gaps | Thematic analysis |
| 6. Report | Write literature review | Structured report |

### Academic Databases

| Database | Coverage | Best For |
|---|---|---|
| Google Scholar | Broad, all fields | General search |
| Semantic Scholar | CS, AI focused | AI/ML research |
| arXiv | Preprints | Latest CS/ML papers |
| IEEE Xplore | Engineering | Systems, hardware |
| ACM Digital Library | Computing | Software, HCI |
| PubMed | Biomedical | Health, bio-AI |
| DBLP | CS bibliography | Author/venue search |

## 3. Experimental Design

### Experiment Types

| Type | Description | When to Use |
|---|---|---|
| A/B test | Compare two variants | Product features |
| Controlled experiment | Isolate variables | Scientific validation |
| Ablation study | Remove components to measure impact | ML model analysis |
| Benchmark | Compare against standard | Performance evaluation |
| Case study | Deep dive on specific instance | Qualitative understanding |
| Survey | Collect opinions/data | User research |

### Experimental Rigor

| Principle | Description |
|---|---|
| Reproducibility | Others can replicate results |
| Statistical significance | Results not due to chance |
| Control variables | Isolate what you're testing |
| Baseline comparison | Compare against known standard |
| Blind evaluation | Evaluators don't know conditions |
| Multiple trials | Repeat to confirm consistency |

## 4. Data Collection and Analysis

### Analysis Methods

| Method | Type | Use Case |
|---|---|---|
| Descriptive statistics | Quantitative | Summarize data |
| Hypothesis testing | Quantitative | Validate claims |
| Regression analysis | Quantitative | Predict relationships |
| Thematic analysis | Qualitative | Identify patterns |
| Meta-analysis | Quantitative | Combine multiple studies |
| Benchmarking | Quantitative | Compare performance |

### Statistical Significance

| Concept | Description | Threshold |
|---|---|---|
| p-value | Probability result is by chance | p < 0.05 |
| Confidence interval | Range of likely true values | 95% CI |
| Effect size | Magnitude of difference | Cohen's d > 0.5 |
| Power | Ability to detect true effect | >0.8 |
| Sample size | Number of observations | Depends on effect size |

## 5. Research Communication

### Paper Structure (CS/ML)

| Section | Content | Length |
|---|---|---|
| Abstract | Summary of entire paper | 150-300 words |
| Introduction | Problem, motivation, contributions | 1-2 pages |
| Related work | Prior art, positioning | 1-2 pages |
| Method | Technical approach, architecture | 2-4 pages |
| Experiments | Setup, results, analysis | 2-4 pages |
| Discussion | Limitations, implications | 0.5-1 page |
| Conclusion | Summary, future work | 0.5 page |

### Research Report (Industry)

```
1. Executive Summary (1 page)
   - Key findings
   - Recommendations
   - Impact

2. Background (1-2 pages)
   - Context, motivation
   - Research question

3. Methodology (1-2 pages)
   - Approach
   - Data sources
   - Analysis methods

4. Findings (3-5 pages)
   - Results
   - Analysis
   - Comparisons

5. Recommendations (1-2 pages)
   - Actionable next steps
   - Implementation plan
   - Resource requirements

6. Appendix
   - Detailed data
   - Additional analysis
```
