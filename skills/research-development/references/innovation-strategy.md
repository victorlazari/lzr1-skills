## 1. Innovation Framework

### Three Horizons of Innovation

| Horizon | Focus | Timeline | Investment | Risk |
|---|---|---|---|---|
| H1: Core | Optimize existing business | 0-12 months | 70% | Low |
| H2: Adjacent | Extend into new areas | 12-36 months | 20% | Medium |
| H3: Transformational | Create new markets | 36+ months | 10% | High |

### Innovation Types

| Type | Description | Example |
|---|---|---|
| Incremental | Small improvements to existing | Feature enhancements |
| Architectural | New combinations of existing tech | Platform redesign |
| Disruptive | New market or value network | New business model |
| Radical | Breakthrough technology | Fundamental new capability |

### Innovation Process

```
Discover → Define → Develop → Deliver → Scale

Discover: Identify opportunities, trends, unmet needs
Define: Frame the problem, set constraints, success criteria
Develop: Prototype, experiment, iterate
Deliver: Build MVP, validate with customers
Scale: Grow, optimize, integrate into core business
```

## 2. Innovation Portfolio

### Portfolio Management

| Category | Criteria | Metrics |
|---|---|---|
| Strategic fit | Alignment with company vision | Strategy score |
| Market potential | Size, growth, accessibility | TAM, growth rate |
| Technical feasibility | Can we build it? | Prototype results |
| Competitive advantage | Differentiation, defensibility | Moat assessment |
| Resource requirements | People, money, time | Investment needed |
| Risk | Technical, market, execution | Risk score |

### Stage-Gate Process

| Gate | Decision | Criteria |
|---|---|---|
| Gate 0 | Explore? | Strategic fit, initial merit |
| Gate 1 | Investigate? | Market potential, technical feasibility |
| Gate 2 | Develop? | Business case, resource availability |
| Gate 3 | Test? | Prototype results, customer feedback |
| Gate 4 | Launch? | Market validation, readiness |
| Gate 5 | Scale? | Unit economics, growth potential |

## 3. Innovation Programs

### Program Types

| Program | Description | Duration | Output |
|---|---|---|---|
| Hackathon | Intensive building event | 1-3 days | Prototypes |
| Innovation lab | Dedicated R&D team | Ongoing | New products |
| Incubator | Internal startup program | 3-6 months | Validated ideas |
| Accelerator | Fast-track promising ideas | 3 months | MVPs |
| Skunkworks | Secret advanced projects | 6-18 months | Breakthroughs |
| Open innovation | External collaboration | Varies | Partnerships |

### Innovation Metrics

| Metric | Description | Target |
|---|---|---|
| Pipeline health | Number of ideas in each stage | Growing |
| Time to market | Idea to launch duration | Decreasing |
| Success rate | % of projects that launch | 10-30% |
| Revenue from new | % revenue from <3yr products | >20% |
| Patent filings | New IP generated | Increasing |
| Experiment velocity | Experiments run per quarter | Increasing |
| ROI | Return on innovation investment | >3x over 5 years |

## 4. Innovation Culture

### Culture Elements

| Element | Practice |
|---|---|
| Psychological safety | Safe to propose wild ideas, fail |
| Time allocation | 10-20% time for exploration |
| Recognition | Celebrate experiments, not just successes |
| Cross-pollination | Mix teams, rotate, expose to new domains |
| External exposure | Conferences, academia, customers |
| Rapid prototyping | Tools and resources for quick experiments |
| Storytelling | Share innovation stories widely |

### Barriers to Innovation

| Barrier | Mitigation |
|---|---|
| Risk aversion | Separate innovation budget, safe-to-fail experiments |
| Short-term focus | Dedicated long-term R&D team |
| Silos | Cross-functional innovation teams |
| Bureaucracy | Lightweight governance for innovation |
| Lack of resources | Protected innovation budget |
| Not-invented-here | Open innovation, external partnerships |

## 5. IP Strategy

### Patent Strategy

| Approach | Description | When |
|---|---|---|
| Offensive | Patent core innovations | Competitive advantage |
| Defensive | Build patent portfolio | Prevent litigation |
| Licensing | License IP for revenue | Monetize non-core IP |
| Open source | Release as open source | Build ecosystem |
| Trade secret | Keep secret, don't patent | Process innovations |

### IP Portfolio Management

| Activity | Frequency | Purpose |
|---|---|---|
| Invention disclosure | Ongoing | Capture new ideas |
| Patentability assessment | Per disclosure | Evaluate merit |
| Filing decisions | Quarterly | Prioritize filings |
| Portfolio review | Annually | Prune, maintain, license |
| Competitive monitoring | Ongoing | Track competitor IP |
| Freedom to operate | Per product launch | Avoid infringement |
